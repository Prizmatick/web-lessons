основні властивості css для роботи з текстом

Властивість font-size

Властивість font-size змінює розмір шрифта.

Розмір шрифту задається за допомогою значення та одиниць виміру(em, px, pt, %)

Синтаксис:

CSS

    p{
        font-size: 20px;
    }

HTML

<p>Hello, world!</p>

Результат:

Hello, world!

Властивість font-weight

Властивість font-weight встановлює насиченість шрифту.

Може приймати наступні значення:

  • normal - нормальна насиченість, те саме, що і 400
  • bold - напівжирний текст, те саме, що і 700
  • lighter - змінює насиченість відносно батьківського стилю в меншу сторону
  • bolder - змінює насиченість відносно батьківського стилю в більшу сторону
  • 100,200,300,400,500,600, 700,800,900 - цифрові значення насищеності, які відрізняються від стандартних

Синтаксис:

CSS

    .bold{
        font-weight: bold;
    }
    .thin{
        font-weight: 100;
    }
    .max{
        font-weight: 900;
    }

HTML

    <p>
        <span class = "bold">Напівжирний текст</span>
        <span class = "thin">найтонший текст</span>
        <span class = "max">найтовщий текст</span>
    </p>

Результат:

Напівжирний текст найтонший текст найтовщий текст

Властивість font-style

Властивість font-style дозволяє вибрати стиль зображення для шрифту.

Може приймати наступні значення:

  • normal - значення за промовчанням встановлює для тексту звичайне зображення шрифту.
  • italic - виділяє текст курсивом.
  • oblique - встановлює похилий шрифт.
  • initial - встановлює значення властивості значення за замовчуванням.
  • inherit - наслідує значення властивості від батьківського елемента.

Синтаксис:

CSS

    .norm{
        font-style: normal;
    }
    .cur{
        font-style: italic;
    }

HTML

<p>
    <span class = "norm">Звичайний</span>
    <span class = "cur">Курсив</span>
</p>

Результат:

Звичайний Курсив

Властивість font-family

Властивість font-family використовується для вибору зображення шрифту.

Може приймати наступні значення:

  • family-name - Назва (ім'я) сімейства шрифтів, наприклад Times, Courier, Arial.
  • generic-family - Базове сімейство шрифтів.

Синтаксис:

CSS

    .def{
        font-family: "Georgia", Serif;
    }
    .gen{
        font-family: serif;
    }

HTML

<p>
    <span class = "def">Текст</span>
    <span class = "gen">Текст</span>
</p>

Результат:

Текст Текст

Властивість color

Властивість color визначає колір тексту елемента.

Синтаксис:

CSS

    .col{
        color: #112233
    }

HTML

<p>
    <span class = "col">Текст</span>
</p>

Результат:

Текст

Властивість text-decoration

Властивість text-decoration додає оформлення тексту в виді підкреслення, перекреслення чи лінії над текстом.

Може приймати наступні значення:

  • line-through - створює перекреслений текст.
  • overline - лінія проходить над текстом.
  • underline - встановлює підкреслений текст.
  • none - скасовує всі ефекти, у тому числі й підкреслення у посилань, яке задано за замовчуванням.

Синтаксис:

CSS

    .th{
        text-decoration: line-through;
    }
    .ov{
        text-decoration: overline;
    }
    .un{
        text-decoration: underline;
    }
    .none{
        text-decoration: none;
    }

HTML

<p>
    <span class = "th">Текст</span>
    <span class = "ov">Текст</span>
    <span class = "un">Текст</span>
    <span class = "none">Текст</span>
</p>

Текст Текст Текст Текст

Властивість text-transform

Властивість text-transform керує перетворенням тексту елемента у великі або великі символи.

Може приймати наступні значення:

  • capitalize - перший символ кожного слова у реченні буде великим.
  • lowercase - всі символи тексту стають малими (нижній регістр).
  • uppercase - всі символи тексту стають великими (верхній регістр).
  • none - не змінює регістр символів.

Синтаксис:

CSS

    .cap{
        text-transform: capitalize;
    }
    .low{
        text-transform: lowercase;
    }
    .up{
        text-transform: uppercase;
    }
    .no{
        text-transform: none;
    }

HTML

<p>
    <span class = "cap">Текст</span>
    <span class = "low">Текст</span>
    <span class = "up">Текст</span>
    <span class = "no">Текст</span>
</p>

Результат:

Текст Текст Текст Текст
